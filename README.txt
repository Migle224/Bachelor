Minios elgesio simuliacijos prototipas
==========================
Norint testuoti projektą reikalinga įsirašyti Unity programinę įrangą, kurią galima persisiųsti iš https://unity3d.com/ . Projektas atliktas naudojant 2017.1.0b4 Unity versiją.
 
Atsidarius projektą, Unity Editor lange įrankių juostoje paleidžiame žaidimo vaizdą. Veikimo režime galime kurti agentus patys naudodami vartotojo sąsają arba paspausti F3 mygtuką klaviatūroje ir paleisti paruoštus simuliacijos testus.  
 
Projektas yra atidaromas Windows operacinėje sistemoje.
 
Plačiau skaityti galima pridėtame rašto darbe.
 
Prototipo autorines teises nustato Vilniaus universiteto dokumentai.
